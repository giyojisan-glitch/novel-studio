# 示例：仙侠短篇《漏斗》

**主角**：林辞，16 岁，青云门外门杂役。先天丹田是"漏斗"——灵气灌进去直接漏掉，百倍苦修仍是凡人。他本想混到 20 岁攒够灵石给瞎眼奶奶买一帖驻颜丹，但师门今天突然以"血脉不洁"为由当众放逐他。他不信命，但也不傻，知道自己下山之后大概率会被灭口——放逐是摆给外人看的体面，真正的命令是"活不过三十里"。

**核心冲突**：外层——林辞必须在三天内穿过三十里的无人区回到林家村，活下来照顾奶奶；内层——他必须决定，当一块会说话的石头在半路叫他"师尊"时，该相信「天上掉馅饼」还是「馅饼里有刀」。赌注是：不修，一定死；修，可能死得更体面。

**世界锚点**：
- 规则 1：丹田等级决定修为上限，漏斗丹田是门派禁忌（只有极少数人能修它，但修它会让修者成为某种东西的"食道"）
- 规则 2：师徒契约以神识为印，解约代价是魂飞魄散——一旦签下，对方死你也活不过一炷香

**基调**：修仙爽文但带黑色幽默。主角嘴碎、自嘲、不装深沉；师尊是被迫收徒的魔尊，两人互相嘲讽的师徒关系是核心卖点。不走纯爽流——每次爽要有代价（修炼一次 = 让师尊多活一点）。

**开场画面**：青云门三千级白石阶最高处，放逐大典的钟响了九下。林辞站在最顶端，身上套着旧灰袍，背着半袋干粮。戒律长老司徒冷没看他，只说了一句"血脉不洁，逐出师门，走"。执法符文化作一只无形的脚，把他从第三千级阶梯一路踹下去。他滚到第一百级时翻身用背滑，滚到第五百级时蜷成一个球，滚到底后站起来拍拍灰，冲钟楼方向笑一句："不送。"
