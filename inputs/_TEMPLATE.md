# Premise 输入模板

> 复制这个文件，改名后填内容。建议 150-400 字。
> 用：`uv run novel-studio init --file inputs/my_novel.md --genre 仙侠`

## 必含五要素

### 1. 主角（具体、有矛盾）
- 身份 / 年龄 / 当前状态
- 一个**具体困境**（不是"被欺负"这种模糊描述）
- 一个**内在矛盾**（想要的 vs 需要的冲突）

### 2. 核心冲突
- 主角 vs 谁 / 什么
- **赌注是什么**（不做会失去什么）
- 为什么**现在**这个冲突爆发（时间锚点）

### 3. 世界锚点（1-2 条硬规则）
- 世界里一个具体的、能约束剧情的规则
- 不要"修仙世界" / "科幻未来"这种大类，要**一条具体的**

### 4. 基调
- 爽文 / 黑色幽默 / 沉重 / 冷峻 / 悬疑
- 或者参考作品：「像《诡秘之主》的压抑感」「像《亮剑》的糙汉气」

### 5. 至少一个具体画面
- 一个开场或高潮的**可视化场景**
- 有具体物件、动作、一句对白
- 这个画面是 AI 的**锚点**——它会围绕这个画面展开整本书
