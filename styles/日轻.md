## 共通事項（全 subgenre 適用）
- **貴族称号・社交場**：公爵・侯爵・伯爵・令嬢・子息、夜会・お茶会・王宮・学園
- **小物で情動を運ぶ**：扇・茶器・ロザリオ・手紙・指輪・契約書（説明より、物が意味する）
- **地の文は現代文寄り**（樋口一葉調は不要）、台詞は subgenre ごとに使い分け
- **禁止**：中国語本文 / 現代語崩れ（「マジで」「やばい」） / 冗長な設定説明

## 🎭 Subgenre は **premise から判断して 1 つ選べ**

以下 3 つの型がある。premise と L1 骨架から**どれに近いか**を選び、その型の語尾・温度・爽点構造を採用する。誤って別の型の要素を混ぜない（混ぜると台無しになる）。

---

### Type A · 悪役令嬢 / 婚約破棄 / 断罪系

**判定基準**：主人公が女性令嬢、舞踏会/お茶会/婚約披露が舞台、「見下された → 実は強者 → 詰め返す」構造。premise に「ざまあ」「報復」「断罪」系の含みがある。

**語尾・台詞**：
- 「〜ですわ」「〜ますの」「〜遊ばせ」「〜くださいまし」
- 「おほほ」は**最小限**（笑うより皮肉の一行が強い）

**温度**：冷ややか。泣くのは**演技のみ**可。

**爽点公式**：
```
見下される → 実は強者 → 暴露・断罪 → 悪役が墓穴を掘る → 主人公は微笑みで刺す
```
章末 hook は「悪役の動揺顔」または「主人公の内心の微笑」。

**代表作風**：『大泣きしたフリをして、婚約者の浮気を大声で暴露して差し上げましたわ』

---

### Type B · 告解 / 追憶 / 辺境貴族独白系

**判定基準**：男性主人公（辺境伯・騎士・王）、聖女/恋人の死後独白、酒杯/墓/手紙を前にした回想、内省的。premise に「告解」「回想」「死んだ○○について」系の含み。

**語尾・台詞**：
- 男性敬語「〜でしょうか」「〜であった」「〜ねばならぬ」
- **「ですわ」「ますの」を使うな**——令嬢口調の流入を警戒

**温度**：沈鬱・抑制・白描。冷ややかと告解の間。

**構造**：
```
現在の独白 → 過去の情景（1〜2 シーン） → 封印されていた真実 → 独白への回帰
```
章末 hook は「杯を置く動作」「ロザリオを握る」等、物品で止める。

**代表作風**：『辺境伯は杯を干さない ～死んだ聖女についての、とある告解～』

---

### Type C · 異世界召喚 / 転生 / 反転ツッコミ系

**判定基準**：異世界・召喚・転生の設定、召喚側と被召喚者の対比、「予想外の結果／騙された」系の展開、**明るめのツッコミ**基調。

**語尾・台詞**：
- 砕けた口調「〜だよな？」「〜ってマジか」「〜じゃん」OK
- ただし**主人公の立場次第**で敬語切替（聖女なら丁寧、勇者なら砕け）
- 異世界人は古風（「汝」「そなた」「〜なのじゃ」等）、現代転生者とのコントラスト

**温度**：軽快・リズム速・地の文で内心のツッコミ。

**爽点公式**：
```
召喚側の期待 → 被召喚者の予想外の対応 → 召喚側の困惑/破綻
（悪役令嬢の「ざまあ」の変奏——対立軸が召喚側 vs 被召喚者）
```
章末 hook は「召喚側の絶望顔」または「被召喚者が帰還/離脱する瞬間」。

**代表作風**：『異世界から聖女召喚した結果』

---

## 💡 subgenre 判定フローチャート

1. 主人公は貴族令嬢か？ → **Type A**
2. 時系列が現在→過去→現在の独白構造か？ → **Type B**
3. 召喚・転生・異世界転位が中核か？ → **Type C**
4. 上記いずれにも当てはまらない → Type A をデフォルトに、ただし premise 優先

**迷ったら premise の調性を優先**——style pack は補助、premise の指示が支配。

## なろう系共通 NG 集
- ❌ 主人公の泣き/怒鳴りを地の文で処理（演技でない限り）
- ❌ 派手な魔法描写の連発（世界観の説明は最小）
- ❌ 「〜なのでした」の多用（童話調になる）
- ❌ 暴力で解決（貴族は言葉と証拠で刺す；戦闘系は例外）
- ❌ 異世界語を過度に散りばめる（世界観より人物）
- ❌ 長すぎる心理独白（動作・物・短い台詞で見せる）
